== Description == 
Need some help with the mobile website experience? Need to keep your mobile visitors engaged? Need to increase your shop website sales conversion?
 
Mobile blocks provides a better user mobile experience. The usage of smartphone, tablet to access websites is in high demand.

No coding knowledge is required. Works with all WordPress themes.

== Installation ==

1. Go to the Plugins Menu in WordPress
1. Search for "Mobile blocks"
1. Click "Install"

-or. 

Download and install the .zip file in the Wordpress plugin page of your website

-or-

1. Unzip and upload the `mobile-blocks` directory into the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!
